Lizenzvereinbarungn:

§1 Benutzen der Mod für Mopacks:
   1. Jeder darf diese Mod für jede art von Modpacks verwenden 
   2. Der Entiwckler dieser Mod behält sich das Recht vor , seine Mod aus Modpacks die ihm aus welchem Grund auch immer missfallen entfernen zu lassen.

§2 Verwenden des Codes für neue Mods oder Addons :
   1. Solange wie an dem Projekt gearbeitet wird darf jeder den Code der Mod (für O.g. Projekte) frei und öffetnlich verwenden.
      1.2 Dies gilt aber nur für den Code welcher auf der offiziellen GitHub Seite der Mod (https://github.com/robinstechprojects/projectBattleLuckyBlocks)
          zu finden ist.
          
§3 Produzieren von YouTube videos in denen die Mod verwendet wird:
   1. Es steht jedem frei eine so genannte "Mod Review" in welcher die Mod Vorgestellt und erklärt wird auf seinem YouTube Kanal zu veröffentlichen
      1.2 In deisen "Mod-Reviews" dürfen keine "falschen" INformation über die Mod , ihr Inhalt oder ihren Entwickler gegeben werden.
          Als falsche Information wird es gewertet wenn man z.B. sagt das die Mod geld kostet oder etwas derartiges was die Vermarktung
          der Mod einschränkt.
